Важно знать:
Програма работает только с целыми числами. Т.е, если где-нибудь не целое деление, то программа выдаст ошибку. Также она не поддерживает переменные степени
1 / 2 + 1 / 2 - error
2 / 2 = 1
2 / 2 * 1 = 1
1 / 2 * 2 - error
(2 * 1) / 2 = 1

x ^ x - error
x ^ x / x ^ x - error
x ^ (x - x) = 1

Программа умеет:
1) Воспринимать правильную запись многочлена (правильная запись очень вольная, т.е. можно ставить любое кол-во пробелов,
опускать знаки, ставить унарные знаки). Ввод может производиться клавишой "Enter" или нажатием на кнопку "Ввод"
2) Указывать на ошибки, если многочлен ввели неправильно
3) Упрощать многочлен
4) Складывать
5) Вычитать
6) Умножать
7) Делить нацело (+ можно смотреть остаток)
8) Брать производную
9) Возводить в степень
10) Копировать многочлены
11) Жить без некоторых своих файлов: data, Settings
12) Сохранять свои настройки и многочлены
13) Выделять всё (множественное выделение)
14) Снимать выделение со всего
15) Разозлиться на пользователя
16) Не залезать в длинную арифметику

Таким образом, программа всегда выдаёт корректный результат, если она ни на что не ругается

Программа имеет:
1) "ИИ" - комментарии к действиям пользователя
2) Звук клавиш
3) Фоновую музыку (3 композиции)
4) "Гибкую" систему настройки

Интересные примеры:
1) 0 ^ 0 = 1
2) 0 ^ (-1) - error (деление на 0)
3) 2 ^ (-2) - error (вещественная арихметика)
4) 1 ^ (-3) = 1
5) (-1) ^ (-1) = -1
6) (-1) ^ (-2) = 1
7) x(2)x = 2x^2
8) -x   (  +3   )   x  (  + 2 )  x ( +1   ) = -6x^3
